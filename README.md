# Redux Cart with Undo Functionality

A simple shopping cart application built with **Redux Toolkit**. Features adding items, removing items, and **Undo** action using a history stack.

## ✨ Features

- Add products to cart
- Remove items from cart
- **Undo** last action (add or remove)
- Real-time total calculation
- Clean and simple UI

## 🛠 Technologies Used

- **React.js**
- **Redux Toolkit** (`createSlice`, actions, reducers)
- `useSelector` & `useDispatch`
- JavaScript (ES6+)
